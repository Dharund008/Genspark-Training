Creating an storage Account, container and uploading a file in it:-

1)open Azure portal > search for storage account > create storage account

2) choose the resource group > name > location > performance > replication > access tier > review + create

* primary service : data lake - for high large data's
                  : azure files - for normal data's(regualr data)
                  : table format - for reusing data's

* then in advanced tab : 
                        Allow enabling anonymous access on invidual container : enable this checkbox!!!!!...

3)after creating storage account: 
        in data storage > containers > Add container 

        * name
        * access level : either blob(public) or container acc to req.....

4) Upload Files (Blobs)
    Open the container
    Click Upload
    Choose a file from your local system (e.g., image.jpg)
    Click Upload

    -- then by clicking the file : copy the url of file and paste it in browser : u can view it.

--- next : static website hosting in azure storage 

    https://learn.microsoft.com/en-us/azure/storage/blobs/storage-blob-static-website

    https://learn.microsoft.com/en-us/azure/storage/blobs/storage-blob-static-website-how-to?tabs=azure-portal


    1) go to storage account > data management > static webstie 
            * enable it
            * asks index doc name and error doc path (404.html)...

            here an container named '$web' will be created and shown in your container list.

    2) then , in your frontend folder :
            terminal > npm run build / ng build --configuration production

    3) this creates an disk folder to work on .. > browser > index.html,styles,main,polyish ...all required files..

    4)move to the $web contianer > upload the files under disk>browser (all necessary files)

    5) next , after uploading it . go again to static website : storage account > data management > static website

        -- will be displaying 2 endpoints primary and secondary 
        --copy any (primary endpoint) and paste it in ur browser

        now your static website is up and running in azure storage...
